# My Portfolio

Bu, mənim şəxsi portfolio layihəmdir. Layihə HTML, CSS və JavaScript istifadə edilərək hazırlanmışdır.

## Layihə Haqqında

- Statik veb sayt
- Responsive dizayn

- Kontakt forması və digər interaktiv elementlər əlavə edilib

## Texnologiyalar

- CSS3
- JavaScript (behavior.js)
