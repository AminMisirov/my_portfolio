Django Blog Proyekti

İyun 2025

Django ilə dinamik blog sistemi yaratdım. İstifadəçi qeydiyyatı, post əlavə etmə, kateqoriya sistemi kimi funksiyalar mövcuddur. Backend və frontend kodları tam inteqrasiya olunub. 


# views.py nümunəsi
from django.shortcuts import render
from .models import Post

def home(request):
    posts = Post.objects.all()
    return render(request, 'home.html', {'posts': posts})

Python Veb Scrapping Skripti

May 2025

Bu Python skripti veb səhifələrdən məlumatları avtomatik yükləyir və işləyir. Requests və BeautifulSoup kitabxanalarından istifadə edir, JSON formatında çıxış verir. 


import requests
from bs4 import BeautifulSoup

url = "https://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")

data = soup.find_all("div", class_="content")
for item in data:
    print(item.text)
